## 🌐 What is Banking System 02?
Advanced banking application with **Spring Boot backend**, persistent SQL storage, and clean REST APIs — the upgraded successor to the JS banking simulator.

---

## ✨ Core Capabilities

### 🏦 Account Management
- Create & manage accounts, balance inquiry, validation

### 💸 Transactions
- Deposit, withdrawal, fund transfer, full history

### 🛡️ Reliability
- Exception handling, input validation, proper HTTP codes

---

## 📡 API Endpoints
```
POST   /account
GET    /account/{id}
POST   /deposit
POST   /withdraw
POST   /transfer
GET    /transactions
```

---

## 🧬 Tech Stack

| Layer | Technology | Purpose |
|-------|-----------|---------|
| 🚪 API | Spring Boot | REST endpoints |
| ⚙️ Logic | Java | Business rules |
| 🗄️ DB | SQL + JPA | Persistent storage |

## 🚀 Quick Start
```bash
git clone https://github.com/Sanmatijain22/Banking_system_02.git
cd Banking_system_02
./mvnw spring-boot:run
```
